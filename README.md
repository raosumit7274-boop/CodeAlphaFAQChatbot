# FAQ Chatbot

## 📌 Project Description
This is a simple FAQ Chatbot built using Python and Streamlit. It answers predefined questions using Natural Language Processing (NLP) with TF-IDF Vectorizer and Cosine Similarity.

## Features
- User-friendly interface
- Answers frequently asked questions
- NLP-based similarity matching
- Fast response

## Technologies Used
- Python
- Streamlit
- Scikit-learn

## How to Run
1. Install the required libraries:
   pip install streamlit scikit-learn

2. Run the application:
   streamlit run app.py

## Output
The chatbot finds the most relevant answer based on the user's question.
